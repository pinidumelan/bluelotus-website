# Blue Lotus

Marketing website for Blue Lotus, a changeover cleaning service for Airbnb and
short-let hosts. Static HTML, CSS and JavaScript with no build step, so it can
be hosted anywhere.

## Pages
- `index.html` — Home
- `services.html` — Services
- `how-it-works.html` — How It Works
- `about.html` — About
- `contact.html` — Contact

## Structure
- `css/styles.css` — all styles
- `js/main.js` — menu, scroll reveal, FAQ and form handling
- `assets/` — logo, favicon and images

## Local preview
Serve the folder over any static server, for example:

```
python -m http.server 5500
```

Then open http://localhost:5500
